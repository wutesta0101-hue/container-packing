# CSV 欄位說明

匯入時請務必使用以下欄位名稱（**全部小寫，順序可以不同但名稱要對**）：

## 欄位定義

| 欄位名稱 | 中文 | 型態 | 必填 | 範例 | 說明 |
|---|---|---|---|---|---|
| `id` | 貨物編號 | 文字 | ✓ | A001 | 唯一識別碼，可用英數字組合 |
| `type` | 類型 | 文字 | ✓ | heavy | 只能是這四種：`normal` / `heavy` / `fragile` / `vip` |
| `length` | 長 (mm) | 數字 | ✓ | 1200 | 必須為正整數，單位毫米 |
| `width` | 寬 (mm) | 數字 | ✓ | 1000 | 必須為正整數,單位毫米 |
| `height` | 高 (mm) | 數字 | ✓ | 800 | 必須為正整數，單位毫米 |
| `weight` | 重量 (kg) | 數字 | ✓ | 800 | 單件重量，公斤 |
| `quantity` | 數量 | 整數 | ✓ | 4 | 同款貨物有幾件 |
| `stackable` | 可堆疊 | 布林 | ✓ | true | 只能填 `true` 或 `false` |

## 類型 (type) 對應的顏色

| type 值 | 中文 | 顯示顏色 | 用途 |
|---|---|---|---|
| `normal` | 一般 | 綠色 | 普通貨物 |
| `heavy` | 重物 | 深藍色 | 重的貨物（會優先排在底部） |
| `fragile` | 易碎品 | 橘色 | 易碎物品（建議搭配 stackable=false） |
| `vip` | VIP 客戶 | 金色 | VIP 客戶的貨物（演算法會優先擺放） |

## 注意事項

1. **第一行必須是欄位標題**，不能有空白行或註解
2. **數字不要加千分位逗號**（寫 `1200` 不要寫 `1,200`）
3. **儲存時使用 UTF-8 編碼**，避免中文 ID 變亂碼（Excel 另存新檔時要選「CSV UTF-8」）
4. **單件尺寸不可超過貨櫃內徑**：
   - 20' 貨櫃：最大 5898 × 2352 × 2393 mm
   - 40' 貨櫃：最大 12032 × 2352 × 2393 mm
   - 40' HQ：最大 12032 × 2352 × 2698 mm

## 完整範例

```csv
id,type,length,width,height,weight,quantity,stackable
A001,heavy,1200,1000,800,800,4,true
B001,fragile,1000,800,600,150,5,false
V001,vip,1400,1100,1000,600,2,true
N001,normal,1000,800,700,250,8,true
```

## 用 Excel 製作 CSV 的步驟

1. 開新的 Excel 檔
2. 第一列填入欄位名稱：`id` `type` `length` `width` `height` `weight` `quantity` `stackable`
3. 從第二列開始填資料
4. **檔案 → 另存新檔 → 檔案類型選「CSV UTF-8 (逗號分隔) (*.csv)」**
5. 拖進系統的 CSV 上傳區即可
