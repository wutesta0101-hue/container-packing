# ============================================================
# .dockerignore — 前端 Docker build 時不要複製進來的檔案
#
# ⚠️ 最重要的一條：node_modules
# 為什麼？因為它可能 200-500MB，COPY 進去會：
#   1. 大幅拖慢 build（複製幾十萬個小檔案很慢）
#   2. 浪費 image 體積（builder 階段會自己 npm ci 重裝）
#   3. 平台不相容（你 Windows 裝的不一定能在 Linux 容器跑）
# ============================================================
